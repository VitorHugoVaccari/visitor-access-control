# Sistema de Controle de Visitantes Técnicos

Sistema web desenvolvido durante o curso técnico no SENAI com foco em controle de acesso de visitantes técnicos em ambientes industriais.

O projeto foi criado para resolver um problema real de rastreabilidade e segurança, permitindo o registro de entrada e saída de prestadores de serviço, controle de acessos ativos e histórico de visitas.

## Funcionalidades

* Autenticação de usuários
* Troca de senha
* CRUD completo de visitantes
* Registro de entrada e saída
* Controle de acessos ativos/finalizados
* Histórico de acessos
* Dashboard de visitantes ativos
* Máscaras de formulário
* Controle de área de destino

## Regras de negócio implementadas

* Um visitante não pode possuir duas entradas abertas simultaneamente
* Controle de status ATIVO e FINALIZADO

## Tecnologias utilizadas

* PHP
* CodeIgniter
* MySQL
* HTML
* CSS
* JavaScript

## Objetivo do projeto

O objetivo do sistema é melhorar o controle de acesso e aumentar a segurança e rastreabilidade de visitantes técnicos em ambientes corporativos e industriais.
